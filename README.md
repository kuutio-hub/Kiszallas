# Költségkalkulátor v3.2.18

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.18)
- **Mentés Nélküli Működés:** Az alkalmazás logikája jelentősen leegyszerűsödött. Az árlisták mentése, létrehozása és törlése megszűnt. A program minden indításkor a gyári árlistákkal indul, melyek díjtételei az adott munkamenet idejére szabadon módosíthatók.
- **Azonnali Visszajelzés:** Az árlistákban végzett módosítások mostantól azonnal látszanak a kalkulációban, megkönnyítve a díjtételek finomhangolását.
- **Javítás:** Az összesített önköltség számítási logikája javítva lett a pontos "Belső" nézet érdekében.

Részletesebb információkért lásd a `changelog.md` fájlt.
