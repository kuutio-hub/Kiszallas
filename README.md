# Költségkalkulátor v3.2.41

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.41)
- **Azonnali Kalkuláció:** Az "Egyéb költségek" tételei mostantól a beírás pillanatában frissülnek a kalkulációban, a korábbi mentéshez kötött viselkedés helyett, ezzel gyorsítva a munkafolyamatot.
- **Egységes Nyomtatási Lábléc:** Az árlisták nyomtatási lábléce teljes mértékben egységesítve lett a fő kalkulációs nyomtatványéval, professzionális és konzisztens megjelenést biztosítva.

Részletesebb információkért lásd a `changelog.md` fájlt.
