
# Költségkalkulátor v3.3.45

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmazás.

## 2. Legutóbbi Változások (v3.3.45)
- **Fejlesztett XLS Export:** A felhasználói visszajelzés alapján az XLS export funkció teljesen megújult. A korábbi többlapos formátum helyett mostantól egy **egyetlen, rendkívül részletes, minden díjtételt és szorzót tartalmazó munkalapot** generál az alkalmazás, amely egy teljes, dokumentum-szerű áttekintést nyújt a kalkulációról.

Részletesebb információkért lásd a `changelog.md` fájlt.
