# Költségkalkulátor v3.2.30

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.30)
- **Felhasználói Élmény Javítása:** A díjtételek szerkesztő felülete logikus csoportokba lett rendezve a jobb átláthatóságért.
- **Bővített Súgó:** A beépített használati útmutató (wiki) jelentősen bővült, hogy minden funkció használata egyértelmű legyen.
- **Stabil Működés:** Az alkalmazás továbbra is a egyszerűsített, munkamenet-alapú díjtétel-kezelést használja, ahol minden tétel ideiglenesen felülírható.

Részletesebb információkért lásd a `changelog.md` fájlt.
