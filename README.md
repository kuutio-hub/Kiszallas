# Költségkalkulátor v3.2.9

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmazás.

## 2. Újdonságok a v3.2.9-ben
- **Teljes Árlistakezelő:** Az alkalmazás ismét lehetővé teszi egyedi árlisták létrehozását, szerkesztését és törlését (CRUD) egy dedikált felületen.
- **Javított Mobil UI:** Az akciógombok ikonjai most már helyesen jelennek meg mobilnézetben.
- **Funkcionális Helyreállítás:** Egy kritikus regresszió javítása, visszaállítva az árlistakezelő teljes működőképességét.
