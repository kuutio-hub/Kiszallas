# Költségkalkulátor v3.2.11

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmazás.

## 2. Újdonságok a v3.2.11-ben
- **Rugalmas Díjtételek:** A szállás, kilométerdíj és emelőgép költségei mostantól a fő felületen, dedikált mezőkben felülírhatók. Ez lehetővé teszi a projektspecifikus költségek egyszerű kezelését anélkül, hogy a mentett árlistákat módosítani kellene.
- **Szétválasztott Logika:** A mentett árlisták már csak az óradíjakat és a kiküldetési díjakat tárolják, míg a változó költségeket az alkalmazás fő nézete kezeli, növelve az átláthatóságot és a rugalmasságot.
