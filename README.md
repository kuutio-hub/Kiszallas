# Költségkalkulátor v3.2.33

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.33)
- **Felhasználói Élmény Javítása:** A díjtételek szerkesztő felülete jelentősen javult. Mostantól minden díjtétel látható és szerkeszthető, az elrendezés pedig sokkal rendezettebb és átláthatóbb.
- **Kritikus Javítás:** Kijavításra került egy súlyos hiba, amely az oldal elrendezésének szétesését okozta. Az alkalmazás most ismét stabil.
- **Bővített Súgó:** A beépített használati útmutató (wiki) továbbra is részletes leírásokkal segíti a használatot.

Részletesebb információkért lásd a `changelog.md` fájlt.
