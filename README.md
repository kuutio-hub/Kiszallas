
# Költségkalkulátor v3.3.44

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmazás.

## 2. Legutóbbi Változások (v3.3.44)
- **Új Funkció: XLS Export:** A kalkuláció eredményei mostantól egy részletes, többlapos XLS fájlba is exportálhatók. A fájl tartalmaz egy vezetői összefoglalót, a teljes költségbontást, az összes bemeneti paramétert és az egyéb költségeket is.
- **UI Bővítés:** Új "Export XLS" gomb került a felületre a funkció egyszerű eléréséhez.

Részletesebb információkért lásd a `changelog.md` fájlt.
