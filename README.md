
# Költségkalkulátor v3.3.46

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmazás.

## 2. Legutóbbi Változások (v3.3.46)
- **Dinamikus Excel Export:** Az export funkció mostantól egy `.xlsx` fájlt generál, amely **Excel függvényeket használ**. A fájl két munkalapot tartalmaz: egy "Kalkuláció" lapot és egy "Díjtételek" lapot. A kalkulációs cellák a díjtételekre hivatkoznak, így a letöltött fájlban a díjak módosítása után a teljes táblázat automatikusan újraszámolódik.

Részletesebb információkért lásd a `changelog.md` fájlt.
