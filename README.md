
# Költségkalkulátor v4.0.0

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmazás.

## 2. Legutóbbi Változások (v4.0.0)
- **Teljes Felhasználói Felület Átalakítás:** Az alkalmazás egy teljesen új, egyoldalas, dokumentum-központú elrendezést kapott. A korábbi lenyitható paneleket és a kétoszlopos nézetet egyetlen, nagy, a nyomtatási képhez hasonló táblázat váltotta fel.
- **Integrált Adatbevitel:** A beviteli mezők mostantól közvetlenül a kalkulációs táblázatban helyezkednek el, lehetővé téve a rendkívül gyors és átlátható adatbevitelt és módosítást.
- **Valós Idejű EUR Átváltás:** A táblázat egy új oszloppal bővült, amely valós időben mutatja minden tétel és a végösszeg EUR megfelelőjét is.
- **Racionalizált Vezérlők:** A globális beállítások (árlista, pénznem, kedvezmény) és a főbb műveletek (nyomtatás, export) egy képernyő tetején rögzített, mindig elérhető sávba kerültek.

Részletesebb információkért lásd a `changelog.md` fájlt.
