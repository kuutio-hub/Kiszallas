# Költségkalkulátor v3.2.19

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.19)
- **UI/UX Fejlesztések:** Számos finomítás történt a felhasználói felületen, beleértve egy újratervezett alsó akciósávot, egy intuitívabb "Egyéb költségek" elrendezést és jobb színkontrasztokat.
- **Funkcionális Egyszerűsítés:** Az árfolyamkezelés egyszerűbb lett a grafikon eltávolításával.
- **Hibajavítás:** A nyomtatási előnézet háttere mostantól minden esetben helyesen, fehéren jelenik meg.
- **Mentés Nélküli Működés:** Az alkalmazás logikája munkamenet-alapú. A díjtételek ideiglenesen módosíthatók, de nincs mentési funkció.

Részletesebb információkért lásd a `changelog.md` fájlt.
