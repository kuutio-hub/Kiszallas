# Költségkalkulátor v3.2.21

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.21)
- **Felhasználói Élmény Javítása:** A felhasználói felületen elvégzett utolsó simítások, beleértve a beviteli mezők javítását és az "Egyéb költségek" szekció finomhangolását.
- **Súgó Hozzáadása:** Egy új, ikonról elérhető Súgó funkció segíti a felhasználókat az alkalmazás használatában és a számítási logika megértésében.
- **Mentés Nélküli Működés:** Az alkalmazás logikája továbbra is munkamenet-alapú. A díjtételek ideiglenesen módosíthatók, de nincs mentési funkció.

Részletesebb információkért lásd a `changelog.md` fájlt.
