# Költségkalkulátor v3.2.23

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.23)
- **Kódkarbantartás:** Javítva lett egy hiba, amely miatt felesleges szöveges tartalom jelent meg az alkalmazás alján.
- **Felhasználói Élmény Javítása:** A felhasználói felületen elvégzett utolsó simítások, beleértve az "Egyéb költségek" szekció igazítását, a beviteli mezők javítását, és a súgó funkció hozzáadását.
- **Mentés Nélküli Működés:** Az alkalmazás logikája továbbra is munkamenet-alapú. A díjtételek ideiglenesen módosíthatók, de nincs mentési funkció.

Részletesebb információkért lásd a `changelog.md` fájlt.
