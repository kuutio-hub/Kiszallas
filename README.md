# Költségkalkulátor v3.2.25

## 1. Projekt Bemutatása
Helyszíni szerelési projektek költségbecslésére szolgáló, modern, reszponzív webalkalmas.

## 2. Legutóbbi Változások (v3.2.25)
- **Kritikus Hibajavítás:** Javítva egy hiba, ami megakadályozta az "Egyéb költségek" megjelenését a nyomtatási előnézetben és a PDF dokumentumokban.
- **Felhasználói Élmény Javítása:** Az árfolyam mellett mostantól látható annak dátuma is. Az "Egyéb költségek" kezelése tovább finomodott az azonnali vizuális visszajelzésekkel.
- **Mentés Nélküli Működés:** Az alkalmazás logikája továbbra is munkamenet-alapú. A díjtételek ideiglenesen módosíthatók, de nincs mentési funkció.

Részletesebb információkért lásd a `changelog.md` fájlt.
